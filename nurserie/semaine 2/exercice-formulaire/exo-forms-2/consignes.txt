Instructions
Créer un nouveau document HTML avec la structure de base et le nommer "cgv.html". Ce document doit être enregistré dans le même dossier que le formulaire précédemment créé.
Dans cgv.html :
Insérer un titre de niveau 1 : Conditions Générales de Vente

Insérer un titre de niveau 2 : Condition 1
Insérer 5 paragraphes avec du faux texte (lorem ipsum)

Répéter l'opération jusqu'à "Condition 5 " pour obtenir 5 blocs contenant chacun 1 titre et plusieurs paragraphes.

Copier ce texte : 
 <style>
        body {
            width: 50%;
            margin: auto;
            padding: 50px 10px;
            font-family: 'Gill Sans', 'Gill Sans MT', Calibri, 'Trebuchet MS', sans-serif;
            font-size: 20px;
        }
        h1 {text-align: center;}
    </style>

Et le coller à l'intérieur de la balise <head></head> du document afin d'ajouter un style CSS à cette page.

Dans le fichier HTML contenant le formulaire, ajouter sur le texte "Conditions Générales de Vente" un lien qui renvoi vers la page des CGV. (le lien doit s'ouvrir dans un nouvel onglet)
